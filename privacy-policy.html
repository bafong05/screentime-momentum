Privacy Policy

Screen Time Momentum

Last updated: April 3, 2026

Screen Time Momentum is designed to help users understand their browsing habits by grouping browsing activity into sessions, tracking intended durations, and generating personal insights.

Information the extension uses

The extension may access and process the following information:

  • visited page URLs
  • page titles and domains
  • timestamps of visits and browsing activity
  • tab and navigation activity needed to group visits into sessions
  • user-created session names
  • user-selected intended durations and notification preferences

How the information is used

This information is used only to provide the extension's core functionality, including:

  • grouping browsing activity into sessions
  • tracking time spent across websites and sessions
  • showing dashboard views such as current session, today, history, and analytics
  • generating insights such as top sites, workflow patterns, extended-session sites, digital footprint paths, and time-of-day trends
  • supporting optional notifications related to session timing, inactivity, and reminders

Data storage

Screen Time Momentum stores browsing and session data locally in the browser using Chrome extension storage. The extension is intended to operate locally for its session tracking and dashboard features.

Data sharing

Screen Time Momentum is not designed to sell user data, share user data with third parties for advertising, or use browsing data for unrelated purposes.

User control

Users can manage their data and experience through the extension by:

  • clearing tracked dashboard data
  • managing notification preferences
  • choosing intended session durations or using No goal

Contact

If you have questions about this policy, please contact the publisher through the Chrome Web Store listing or the project's public repository.
